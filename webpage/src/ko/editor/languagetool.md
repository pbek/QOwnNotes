# 언어 도구

QOwnNotes는 노트 편집기에서 **문법 및 스타일 확인** 옵션으로 로컬 또는 원격 [언어 도구](https://languagetool.org/) 서비스를 사용할 수 있습니다.

기존 [맞춤법 검사](spellchecking.md) 지원을 대체하는 대신 보완합니다.

## 기능

- 감지된 문법, 스타일, 구두점 및 활자 서체 배열 문제에 대한 색상 밑줄
- 교체를 위한 컨텍스트 메뉴 제안
- '더 많은 정보...' 추가 문서를 제공하는 규칙 링크
- 다시 보고 싶지 않은 규칙에 대한 '이 규칙을 무시하세요' 행동
- **맞춤법** 메뉴에서 '언어 도구로 문법 확인'으로 빠르게 전환

## 설정

'설정 -> 편집기'를 열고 **언어 도구** 섹션을 구성합니다.

- '언어 도구로 문법 및 스타일 검사 사용함'을 켭니다
- '서버 URL' 설정
  - 로컬 서버는 'http://localhost:8081'과 같은 기능을 사용할 수 있습니다
  - QOwnNotes는 해당 서버에서 '/v2/check' 엔드포인트를 자동으로 사용합니다
- '언어' 선택
  - '자동 감지' 사용 가능
  - 필요한 경우 사용자 지정 언어 코드를 입력할 수도 있습니다
- 언어 도구 서비스에 'API 키'가 필요한 경우 선택적으로 'API 키'를 추가합니다
- 요청을 보내기 전에 QOwnNotes가 입력한 후 대기하는 시간을 제어하기 위해 '확인 지연'을 조정합니다
- 확인해야 할 범주 선택
  - `맞춤법`
  - `문법`
  - `스타일`
  - `구두법`
  - `활자 서체 배열`

'연결 테스트'를 사용하여 구성된 서비스에 도달할 수 있는지 확인합니다.

## 작동 방식

- QOwnNotes는 전체 문서 대신 보이는 편집기 블록을 한 번에 확인합니다
- 빈 줄, 제목 및 코드 블록을 건너뛰었습니다
- 요청이 삭제되므로 모든 키 입력 시 서비스가 조회되지 않습니다
- 결과는 Markdown 편집기에 인라인으로 표시됩니다

## 노트

- 언어 도구 지원은 선택 사항이며 빌드에서 활성화되는 기능에 따라 달라집니다
- 서비스에 연결할 수 없으면 QOwnNotes에 경고가 표시되고 다시 사용할 수 있을 때까지 확인이 중지됩니다
- 로컬 자체 호스팅 언어 도구 서버 또는 다른 호환 언어 도구 엔드포인트와 함께 사용할 수 있습니다
